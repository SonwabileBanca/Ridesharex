Renter User Journey

Complete flow from discovery to rental completion

DISCOVERY & REGISTRATION

1
Platform Discovery
User discovers RideShare through various channels
Social Media Google Search Word of Mouth
Discovery Process:
  • User sees RideShare advertisement
  • Friend recommends the platform
  • Google search for car rentals
  • User clicks on website link
2
Land on Website
User arrives at RideShare platform
Browse Vehicles
Landing Experience:
  • Homepage loads with featured vehicles
  • Search bar prominently displayed
  • User can browse without registration
  • Platform showcases available vehicles
3
User Registration
Create account and complete verification
Sign Up
Registration Process:
  • Enter personal details (name, email, phone)
  • Create secure password
  • Verify email address
  • Upload driver's license
  • Take selfie for verification

SEARCH & BOOKING

4
Vehicle Search
Search for available vehicles
Enter Criteria
Search Process:
  • Enter pickup location and dates
  • Select vehicle type and budget
  • Apply filters (price, features)
  • View search results
5
Select Vehicle
Choose preferred vehicle
View Details
Selection Process:
  • Review vehicle photos and specs
  • Check owner profile and ratings
  • Read reviews and comments
  • Compare pricing and features
6
Booking Process
Complete booking and payment
Book Now
Booking Steps:
  • Review rental details and pricing
  • Add insurance coverage if desired
  • Enter payment information
  • Process security deposit
  • Submit booking request

RENTAL EXPERIENCE

7
Vehicle Pickup
Pick up the rented vehicle
Meet Owner
Pickup Process:
  • Receive pickup instructions
  • Navigate to pickup location
  • Meet owner and inspect vehicle
  • Document vehicle condition
  • Sign digital rental agreement
  • Receive keys and documents
8
Active Rental Period
Use the vehicle during rental
Enjoy Ride Report Issue
During Rental:
  • Use vehicle according to terms
  • Follow safety guidelines
  • Keep vehicle clean and secure
  • Contact support if needed
  • Report any issues immediately
9
Vehicle Return
Return the vehicle
Return Vehicle
Return Process:
  • Return to designated location
  • Meet owner for inspection
  • Return keys and documents
  • Complete return inspection
  • Resolve any damage issues
10
Post-Rental
Complete rental experience
Rate & Review
Completion Process:
  • Rate owner and vehicle experience
  • Write detailed review
  • Receive final receipt
  • Security deposit released
  • Consider future rentals
